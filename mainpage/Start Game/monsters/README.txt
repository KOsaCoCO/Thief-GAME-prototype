Save the three monster images into this folder using these exact filenames:

  monster1.png   -> the dark spiky figure with star eyes (image 1)
  monster2.png   -> the red gear creature with eyes (image 2)
  monster3.png   -> the pink character with circular ears (image 3)

On each page load, one of these is picked at random and shown
above the middleground image (between middleground and foreground).

To add more monsters later: drop additional PNGs into this folder
and add their filenames to the MONSTER_FILES array in "Start Game.js".

If your files are .jpg or .webp instead of .png, either rename them
or update the MONSTER_FILES array in "Start Game.js" to match.
